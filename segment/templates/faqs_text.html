What is HoneyBox.ml?

HoneyBox.ml is a software tool designed to help you in the image sementation and tagging problem. Getting a set of labeled segments from your own images hasn't never been easier.

Why this name?

Honeybox.ml is a wordgame combining the term "honeypot project" used by Andrew Ng in a video talking about what to do next in a machine learning project and the bounding boxes that a user does to capture the segments :)

Do you have a tutorial or video on how to use this software?

Not yet but,please, give it a try, you will be surprised on how easy is to get your training set in your matlab/octave box ;)

What I will obtain using this tool?

For each training set that you segment you can get a zipfile with a folder for every tag filled with the images tagged by that tag or a matlab file with the following variables:

  • X is a matrix with a row for every image.
  • labels is an ordered list of strings with the tag names corresponding to variable Y.
  • A binary matrix where the ith row indicates whether the ith image of X is tagged with tag j or not.

Do I need to create a user account to use this software?

You can get the first contact with the tool just clicking the button demo but, if you wan't to have your own private training sets a user account is necessary.
